# CamusDB Documentation

This repository contains the Docusaurus site published at
https://camusdb.github.io/.

The source project lives at https://github.com/camusdb/camusdb.

## Install

```bash
npm install
```

## Develop

```bash
npm run start
```

## Build

```bash
npm run build
```

The static site is generated in `build/`.

## Deploy

```bash
npm run deploy
```
